## 1. PROJE GENEL BAKIŞ

### 1.1 Proje Tanımı

**ÖZKAN-OS**, tamamen sıfırdan yazılmış, **GRUB veya Linux çekirdeği kullanmayan**, **Rust** ile geliştirilmiş, **7 farklı işlemci mimarisini** destekleyen, **yerli ve milli** bir işletim sistemidir.

### 1.2 Temel Felsefe

| Prensip | Açıklama |
|---------|----------|
| **Privacy by Design** | Kullanıcı verisi asla toplanmaz, sıfır telemetri |
| **Secure by Default** | Tüm harici kod sandbox içinde çalışır |
| **Mavi Ekran Yok** | Her hata izole edilir, sistem çalışmaya devam eder |
| **Yeniden Başlatma Yok** | Livepatch ile tüm güncellemeler anında uygulanır |
| **Telif Hakkı Yok** | Tüm kodlar, temalar, fontlar özgün veya açık lisanslı |

### 1.3 Teknik Özellikler

| Alan | Değer |
|------|-------|
| **Ana Dil** | Rust (`#![no_std]`) |
| **Yardımcı Diller** | C (sürücüler), Assembly (boot/HAL) |
| **Varsayılan Dil** | Türkçe (anlık değiştirilebilir) |
| **Mimari Desteği** | 486DX4, x86, x86_64, ARM32, AArch64, RISC-V, MIPS |
| **Toplam Tahmini Satır** | ~200.000+ satır (tamamlandığında) |
| **Minimum RAM** | 8 MB (486DX4) |
| **Önerilen RAM** | 256 MB+ |
| **Disk Alanı** | 50 MB (minimum kurulum) |

### 1.4 Performans Hedefleri

| Test | ÖZKAN-OS Hedefi | Windows 11 | Ubuntu 24 |
|------|----------------|------------|-----------|
| Boot süresi | < 1.8 saniye | ~9.2 sn | ~8 sn |
| Uygulama açılışı | < 0.09 saniye | ~0.31 sn | ~0.25 sn |
| Dosya kopyalama (1 GB) | ~0.42 saniye | ~1.1 sn | ~0.9 sn |

## 3. BOOTLOADER MİMARİSİ

### 3.1 Genel Boot Zinciri

```
BIOS/U-Boot/OpenSBI → Stage1 (MBR) → Stage2 → PM32 → LM64 → Kernel
```

### 3.2 Mimariye Özel Boot Yöntemleri

| Mimari | Boot Metodu | Dosya |
|--------|-------------|-------|
| **x86_64** | BIOS MBR + INT 13h | `stage1_mbr.asm`, `stage2_loader.asm` |
| **x86 (486DX4)** | BIOS MBR + CHS fallback | `stage1_mbr.asm`, `stage2.asm` |
| **AArch64** | EL3→EL2→EL1 geçişi | `boot.S`, `exception_vectors.S` |
| **RISC-V** | OpenSBI S-mode | `boot.S`, `sbi.S` |
| **MIPS** | U-Boot/YAMON | `boot.S`, `tlb.S` |
| **ARM32** | ATAGS/DTB | `boot.S`, `l2cache.S` |

### 3.3 Bootloader Özellikleri

| Özellik | Açıklama | Durum |
|---------|----------|-------|
| **GRUB Kullanımı** | KESİNLİKLE YASAK | ✅ |

| **MBR (512 byte)** | Stage1, boot signature 0xAA55 | ✅ |
| **LBA Disk Okuma** | INT 13h AH=42h | ✅ |

## 4. KERNEL MİMARİSİ

### 4.1 Kernel Yapısı

```
kernel/
├── ARCH/           ← Mimariye özel (x86_64, aarch64, riscv64, mips, arm, x86)
├── COMMON/         ← Mimari bağımsız (scheduler, memory, process, ipc, fs, net)
├── HAL/            ← Donanım soyutlama katmanı (trait'ler)
├── DRIVERS/        ← Sürücüler (USB, storage, net, gpu)
├── GRAPHICS/       ← Grafik (framebuffer, bootlogo)
└── SYSCALL/        ← Sistem çağrıları
```

### 4.2 Kernel Entry Point

```rust
#[no_mangle]
pub extern "C" fn kernel_main() -> ! {
    // 1. Mimari başlat
    arch::init_arch();
    // 2. HAL başlat
    hal::init_hal();
    // 3. Bellek yöneticisi başlat
    common::memory::init();
    // 4. Zamanlayıcı başlat
    common::scheduler::init();
    // 5. Grafik sistemi başlat
    graphics::init();
    // 6. Kesme sistemi başlat
    arch::interrupt::init();
    // 7. Kullanıcı alanı başlat
    common::process::init_userland();
    // 8. Ana döngü
    loop { arch().halt(); }
}
```

### 4.3 Kernel Boot Adımları

| Adım | Açıklama | Serial Çıktı |
|------|----------|--------------|
| 1 | Mimari başlatılıyor | `A` |
| 2 | HAL başlatılıyor | `H` |
| 3 | Bellek yöneticisi başlatılıyor | `M` |
| 4 | Zamanlayıcı başlatılıyor | `S` |
| 5 | Grafik sistemi başlatılıyor | `G` |
| 6 | Kesme sistemi başlatılıyor | `I` |
| 7 | Kullanıcı alanı başlatılıyor | `U` |
| 8 | Masaüstü hazırlanıyor | `D` |

## 5. HAL (HARDWARE ABSTRACTION LAYER)

### 5.1 HAL Trait'leri

```rust
pub trait SerialPort {
    fn putchar(&mut self, c: u8);
    fn getchar(&mut self) -> Option<u8>;
    fn puts(&mut self, s: &str);
}

pub trait Framebuffer {
    fn width(&self) -> u32;
    fn height(&self) -> u32;
    fn put_pixel(&mut self, x: u32, y: u32, color: u32);
    fn fill_rect(&mut self, x: u32, y: u32, w: u32, h: u32, color: u32);
    fn clear(&mut self, color: u32);
    fn swap_buffers(&mut self);
}

pub trait Input {
    fn mouse_position(&self) -> (i32, i32);
    fn mouse_pressed(&self) -> bool;
    fn keyboard_key(&self) -> Option<u8>;
}

pub trait PciAccess {
    fn read_config32(&self, bus: u8, dev: u8, func: u8, offset: u8) -> u32;
    fn write_config32(&mut self, bus: u8, dev: u8, func: u8, offset: u8, value: u32);
}

pub trait Timer {
    fn current_ticks(&self) -> u64;
    fn sleep_ms(&mut self, ms: u64);
}
```

| aarch64 | PL011 | SimpleFB | PL050 | ECAM | Generic Timer |
| riscv64 | UART | SimpleFB | PS/2 | ECAM | CLINT |

## 3. ⚡ **SCHEDULER OPTİMİZASYONU**

### 3.1 Tickless Scheduler

```rust
// Şu an: sabit tick (1000Hz, her 1ms kesme)
// Öneri: Tickless (sadece ihtiyaç olduğunda kesme)

pub struct TicklessScheduler {
    next_event: u64,  // Bir sonraki olay zamanı
}

impl TicklessScheduler {
    pub fn schedule(&mut self) {
        let now = get_time();
        if now >= self.next_event {
            // Sadece zamanı gelen olayları işle
            self.process_due_events();
            self.next_event = self.find_next_event();
            self.set_timer(self.next_event - now);
        }
        // Hiç olay yoksa CPU'yu durdur (HLT)
        if self.next_event == u64::MAX {
            arch().halt();
        }
    }
}
```

```rust
pub struct PageCache {
    pages: LruCache<u64, Page>,
    hit_rate: AtomicU8,
}

impl PageCache {
    pub fn read_page(&mut self, offset: u64) -> &Page {
        if let Some(page) = self.pages.get(&offset) {
            self.hit_rate.fetch_add(1, Ordering::Relaxed);
            return page;  // Cache hit: ~10ns
        }
        // Cache miss: ~1ms (disk access)
        let page = self.read_from_disk(offset);
        self.pages.insert(offset, page);
        &page
    }
}
```

```rust
            self.current = (self.current + 1) % MAX_CTX;
            self.restore_context();
```

## 8. 🧵 **IPC OPTİMİZASYONU**

### 8.1 Lock-free Queues

```rust
// Şu an: Mutex ile korunan queue
// Öneri: Lock-free ring buffer

pub struct LockFreeQueue<T, const N: usize> {
    buffer: [UnsafeCell<T>; N],
    head: AtomicUsize,
    tail: AtomicUsize,
}

impl<T, const N: usize> LockFreeQueue<T, N> {
    pub fn push(&self, item: T) -> Result<(), T> {
        let tail = self.tail.load(Ordering::Acquire);
        let next = tail.wrapping_add(1) % N;
        
        if next == self.head.load(Ordering::Acquire) {
            return Err(item);  // Full
        }
        
        unsafe { *self.buffer[tail].get() = item; }
        self.tail.store(next, Ordering::Release);
        Ok(())
    }
}
```

### 22.1 Value Prediction

```rust
// Öneri: Değer tahmini ile spekülatif çalıştırma

pub struct ValuePredictor {
    last_values: HashMap<MemoryAddress, u64>,
    stride_table: HashMap<MemoryAddress, i64>,
}

impl ValuePredictor {
    pub fn predict_load(&mut self, addr: MemoryAddress) -> Option<u64> {
        let last = self.last_values.get(&addr);
        let stride = self.stride_table.get(&addr);
        
        if let (Some(&last), Some(&stride)) = (last, stride) {
            // Stride prediction
            Some(last.wrapping_add(stride as u64))
        } else if let Some(&last) = last {
            // Last value prediction
            Some(last)
        } else {
            None
        }
    }
    
    pub fn speculative_execute(&mut self, addr: MemoryAddress) -> Result<u64> {
        if let Some(predicted) = self.predict_load(addr) {
            // Spekülatif olarak çalıştır
            let speculative_result = self.execute_speculative(addr, predicted);
            
            // Gerçek değeri yükle
            let actual = self.load_actual(addr);
            
            if speculative_result == actual {
                self.prediction_hits += 1;
                Ok(actual)
            } else {
                self.prediction_misses += 1;
                self.rollback();
                Err(())
            }
        } else {
            self.load_actual(addr)
        }
    }
}
```

```rust
impl TransactionalMemory {
    pub fn transaction<R>(&self, f: impl FnOnce() -> R) -> Result<R, TxError> {
        if !self.has_rtm {
            return self.fallback_lock(f);
        }
        
        unsafe {
            asm!("xbegin 1f", options(nomem, nostack));
            // Transaction body
            let result = f();
            asm!("xend", options(nomem, nostack));
            Ok(result)
        }
        
        // Transaction abort fallback
        // 1f:
        // xend
        // lock-based fallback
    }
}
```

### 23.2 Software Transactional Memory (STM)

```rust
// Öneri: Software transactional memory for complex operations

pub struct StmTransaction {
    read_set: HashSet<MemoryAddress>,
    write_set: HashMap<MemoryAddress, u64>,
    undo_log: Vec<MemoryAddress>,
}

impl StmTransaction {
    pub fn read<T>(&mut self, addr: &Atomic<T>) -> T {
        let addr_val = addr as *const Atomic<T> as MemoryAddress;
        self.read_set.insert(addr_val);
        unsafe { (addr_val as *const T).read() }
    }
    
    pub fn write<T>(&mut self, addr: &Atomic<T>, value: T) {
        let addr_val = addr as *const Atomic<T> as MemoryAddress;
        
        if !self.write_set.contains_key(&addr_val) {
            // Save old value for rollback
            let old = unsafe { (addr_val as *const T).read() };
            self.undo_log.push(addr_val);
        }
        
        self.write_set.insert(addr_val, value);
    }
    
    pub fn commit(&mut self) -> bool {
        // Validation: read_set unchanged?
        for addr in &self.read_set {
            if self.has_changed(*addr) {
                self.rollback();
                return false;
            }
        }
        
        // Commit writes
        for (addr, value) in &self.write_set {
            unsafe { (addr as *mut u64).write(*value); }
        }
        true
    }
}
```

```rust
    bounding: Capability,    // Maksimum yetki seti
}

impl SecureProcess {

        self.effective.bits &= !cap.bits;
        self.permitted.bits &= !cap.bits;
        self.bounding.bits &= !cap.bits;

    }
```

```rust
    verify: bool,

impl SecureWipe {
    pub fn wipe_file(&self, path: &Path) -> Result<(), IoError> {
        let file = File::open(path)?;
        let size = file.metadata()?.len();
        
        for pass in 0..self.passes {
            let pattern = self.get_pattern(pass);
            self.write_pattern(&file, pattern, size)?;
            
            if self.verify {
                self.verify_pattern(&file, pattern, size)?;
            }
        }
        
        fs::remove_file(path)?;
        Ok(())
    }
    
    fn get_pattern(&self, pass: u32) -> u8 {
        match pass {
            0 => 0x00,
            1 => 0xFF,
            2..=4 => 0x55,  // 01010101
            5..=7 => 0xAA,  // 10101010
            8..=34 => random::<u8>(),
            _ => 0x00,
        }
    }
}
```

```rust
pub struct QuantumRng {
    has_rdrand: bool,
    has_quantum: bool,  // Quantum RNG hardware
}

impl QuantumRng {
    pub fn random_bytes(&self, buf: &mut [u8]) {
        if self.has_quantum {
            // Use quantum RNG (entropy from quantum phenomena)
            self.quantum_read(buf);
        } else if self.has_rdrand {
            // Fallback to hardware RNG
            for chunk in buf.chunks_mut(8) {
                let val = self.rdrand64();
                chunk.copy_from_slice(&val.to_ne_bytes());
            }
        } else {
            // Software fallback with jitter entropy
            self.jitter_entropy(buf);
        }
    }
    
    fn jitter_entropy(&self, buf: &mut [u8]) {
        // CPU timing jitter as entropy source
        for byte in buf {
            let start = rdtsc();
            let end = rdtsc();
            *byte = (end - start) as u8;
        }
    }
```

```rust
        // 1. Verify signature
        let signature = self.extract_signature(image);
        if !self.public_key.verify(image, &signature) {

        }

        if self.signature_db.is_revoked(&signature) {

        let update_sig = self.extract_update_signature(new_key.as_bytes());
        if self.public_key.verify(new_key.as_bytes(), &update_sig) {
            self.public_key = new_key;
            Ok(())
        } else {
            Err(BootError::InvalidSignature)
        }
    }
```

```rust
// Öneri: Shamir's Secret Sharing

pub struct SecretSharing {
    threshold: u32,  // k of n shares needed
    total: u32,      // total shares
}

impl SecretSharing {
    pub fn split_secret(&self, secret: &[u8]) -> Vec<Share> {
        // Shamir's Secret Sharing
        let polynomial = self.generate_polynomial(secret);
        let mut shares = Vec::new();
        
        for i in 1..=self.total {
            let share = self.evaluate_polynomial(&polynomial, i);
            shares.push(Share::new(i, share));
        }
        
        shares
    }
    
    pub fn combine_shares(&self, shares: &[Share]) -> Vec<u8> {
        // Lagrange interpolation
        let mut result = vec![0u8; SECRET_SIZE];
        
        for i in 0..self.threshold {
            let share = &shares[i as usize];
            let mut numerator = 1.0;
            let mut denominator = 1.0;
            
            for j in 0..self.threshold {
                if i != j {
                    numerator *= -shares[j as usize].x;
                    denominator *= share.x - shares[j as usize].x;
                }
            }
            
            let lagrange = numerator / denominator;
            self.add_to_result(&mut result, share.y, lagrange);
        }
        
        result
    }
}
```
